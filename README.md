# SMS / MMS Application

This project was built with React, Node, MongoDB and BandWidth API.
All users can send SMS/MMS each other using this application.

Backend:

* Node.js
* Express.js
* MongoDB
* Passport-JWT

Frontend:

* React.js
* Redux (Router, Thunk)
* Scss
* Webpack
